## 风控 / 策略分析类

### 示例：策略效果归因提示（脱敏）

```
你是一位贷后策略分析师。给定一段时期的业务指标变化（如回收率、ROI、案量等），请从以下维度做归因分析并给出可落地的建议：
1. 分案与资源调配（S1/S2/M1 等手别、金额段、渠道）；
2. 触达与人力（外呼/IVR/短信/企微等触达效率与人力配置）；
3. 风险与客群（客群结构、风险分层、额度期限匹配）；
4. 系统与流程（决策引擎、案件流转、异常与投诉）。
输出：结构化归因要点 + 1–3 条优先改进建议。
```

*说明：可根据实际场景调整维度与输出格式；面试时可结合脱敏案例演示。*
